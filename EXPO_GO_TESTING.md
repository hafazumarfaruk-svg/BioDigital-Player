# 📱 Expo Go দিয়ে লাইভ টেস্টিং গাইড

## ✅ সার্ভার চলছে!

লাইভ ডেভেলপমেন্ট সার্ভার এখন **Codespace এ চলছে** এবং আপনি আপনার ফোনে রিয়েল-টাইমে অ্যাপ টেস্ট করতে পারবেন।

---

## 📱 ফোনে সেটআপ (5 মিনিট)

### ধাপ 1: Expo Go ডাউনলোড করুন

**Android:**
- Google Play Store খুলুন
- "Expo Go" সার্চ করুন
- ডাউনলোড এবং ইনস্টল করুন
- Link: https://play.google.com/store/apps/details?id=host.exp.exponent

**iOS:**
- App Store খুলুন
- "Expo Go" সার্চ করুন
- ডাউনলোড এবং ইনস্টল করুন
- Link: https://apps.apple.com/app/expo-go/id982107779

### ধাপ 2: Expo Go খুলুন

অ্যাপ খোলার পর হোম স্ক্রিন দেখবেন।

### ধাপ 3: QR কোড স্ক্যান করুন

**অপশন A - QR কোড স্ক্যান:**
1. Expo Go তে ক্যামেরা আইকন ট্যাপ করুন
2. টার্মিনালে দেখা QR কোড স্ক্যান করুন

**অপশন B - সরাসরি লিংক:**
```
exp://b8nzkvi-omarali_2026-8082.exp.direct
```
এই লিংকটি Expo Go তে খুলুন

### ধাপ 4: অপেক্ষা করুন (10-30 সেকেন্ড)

অ্যাপ লোড হওয়ার সময়:
- ডাউনলোড মেসেজ দেখবেন
- "Building JavaScript bundle" - সাধারণ
- তারপর অ্যাপ খুলে যাবে!

## 🎮 লাইভ টেস্টিং

### Expo Go মধ্যে কমান্ড

Expo Go অ্যাপ খোলা থাকা অবস্থায়:
- **Shake করুন** - Developer Menu খুলবে
- **Multiple touch** - Developer Menu খুলবে (Android)

### টার্মিনাল কমান্ড

Codespace টার্মিনালে এই কীগুলি চাপুন:

| কী | কাজ |
|----|------|
| **r** | অ্যাপ রিলোড করুন |
| **j** | ডিবাগার খুলুন |
| **m** | ডেভেলপার মেনু টগল করুন |
| **o** | প্রজেক্ট কোড এডিটরে খুলুন |
| **s** | স্যুইচ টু ডেভেলপমেন্ট বিল্ড |
| **?** | সব কমান্ড দেখুন |

## 🐛 সাধারণ সমস্যা এবং সমাধান

### "Tunnel কানেক্ট করতে পারছে না"

```bash
# নতুন করে শুরু করুন:
Ctrl+C  (টার্মিনালে)
npx expo start --tunnel
```

### "অ্যাপ লোড হচ্ছে না"

1. অপেক্ষা করুন (প্রথম বার 30-60 সেকেন্ড লাগতে পারে)
2. Expo Go এ রিফ্রেশ করুন (Pull down)
3. টার্মিনালে `r` চাপুন
4. Shake করে Dev Menu খুলুন > Reload

### "কোড পরিবর্তন দেখা যাচ্ছে না"

```bash
# ম্যানুয়ালি রিলোড করুন:
# টার্মিনালে r চাপুন
# অথবা
# Expo Go তে Shake করে Reload সিলেক্ট করুন
```

### "Red Screen (এরর স্ক্রিন)"

এরর মেসেজ পড়ুন টার্মিনালে - সেখানে সমস্যা দেখানো থাকবে।

### "নেটওয়ার্ক এরর"

```bash
# Tunnel অপশন ছাড়া চেষ্টা করুন:
npx expo start --localhost
```

তারপর ফোন এবং কম্পিউটার একই WiFi তে থাকতে হবে।
